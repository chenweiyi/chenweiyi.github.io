  3. 理解package.json文件中的~和^
    ~ 表示匹配中间数字，比如：~1.2.3将会匹配所有1.2.x版本，但是不会匹配1.3.0
    ^ 表示匹配第一个数字，比如：^1.2.3将会匹配所有1.x.x版本，但是不会匹配2.0.0
    总结一下：
    ~1.2.3: >= 1.2.3 <** 1.3.0
    ^1.2.3: **>=     1.2.3 < 2.0.0
    ^符号时npm安装插件的默认符号
